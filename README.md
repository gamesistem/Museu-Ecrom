Baixe e instale o .deb em seu Linux Debian/Ubuntu Based.

Baixe e instale o .exe em seu Windows 10/11.

Baixe e instale o .apk em seu Android 13+.

Com o Launcher instalado será possivel baixar de forma mais rápida e organizada as versões do museu.

Aviso: Se Por Algum Motivo não for possivel achar o Launcher para seu dispositivo em releases nem no Main significa que ele ainda não possui uma versão para tal dispositivo.
